If you have a .zip file, you must extract it first. Chrome cannot load the extension directly from the ZIP archive.

Open your browser and navigate to the extensions management area:

Chrome: Type chrome://extensions/ in the address bar and press Enter.

Brave: Type brave://extensions/ in the address bar and press Enter.

Alternatively: Click the Menu (three dots or lines) > Extensions > Manage Extensions.

 # Enable Developer Mode

In the top-right corner of the Extensions page, you will see a toggle switch labeled Developer mode.

Switch this toggle to ON.

Once enabled, a new sub-header will appear with three buttons: Load unpacked, Pack extension, and Update.

 Load the Extension
 
Click the Load unpacked button.

A file explorer window will open. Navigate to the folder where you extracted your extension.

Select the folder (the one containing the manifest.json) and click Open or Select Folder.

The extension should now appear in your list of installed extensions and be ready for use.

Important Notes
Keep the folder: Do not delete or move the folder after loading it. The browser "links" to that specific location. If you move it, the extension will break.
